# 📖 Overview

**ft_transcendence** is the final full-stack web project of the **42 Common Core**, transforming the classic Pong game into a modern online gaming platform.

The application combines **secure authentication**, **real-time multiplayer gameplay**, **live chat**, **matchmaking**, **player profiles**, and **match statistics** into a responsive web experience powered by modern web technologies.
