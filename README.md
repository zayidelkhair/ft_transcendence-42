## 📖 Overview

**ft_transcendence** is a modern full-stack web application centered around a **real-time multiplayer Pong** experience. It combines secure authentication, interactive gameplay, live chat, matchmaking, player profiles, and match history into a single platform. The project showcases modern web technologies, scalable architecture, and collaborative software engineering practices.
